

# 登录请求

##### 简要描述	

```cmd
用户登录接口	
```

##### 请求URL

```cmd
http://127.0.0.1/api/login
```

##### 请求方式

```cmd
POST
```

##### 参数

| 参数名   | 必选 | 类型   | 说明     |
| -------- | ---- | ------ | -------- |
| username | 是   | string | 用户名称 |
| pwd      | 是   | string | 用户密码 |

##### 返回参数说明

| 参数名 | 类型   | 说明                              |
| ------ | ------ | --------------------------------- |
| msg    | string | 登录信息提示                      |
| status | num    | 登录是否成功。200: 成功，256,错误 |
| data   | object | 服务器响应的数据                  |
|        |        |                                   |
